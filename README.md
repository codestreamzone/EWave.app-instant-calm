# EWave Landing Page

The official landing page for EWave - an instant relief mobile app designed to reduce stress, anxiety, and emotional overload.

## Overview

This static website serves as the informational hub and waitlist portal for the upcoming EWave mobile application.

## Features

-   **Modern UI:** Features a premium design with glassmorphism effects and smooth scroll animations.
-   **Waitlist System:** Integrated email signup form for users to join the launch notification list.
-   **Responsive Layout:** Fully optimized experience across desktop, tablet, and mobile devices.

## Tech Stack

-   HTML5
-   CSS3
-   Vanilla JavaScript

## Running Locally

To view the landing page locally:

1.  Clone the repository.
2.  Open `index.html` directly in your browser or run a simple local server:

```bash
npx serve
```
